# Getting Started

Check if Docker is installed.
```bash
which docker
```

Make sure you are part of the docker group.
```bash
sudo usermod -aG docker USERNAME
```

Access docker help.
```bash
docker --help
```

Access docker help for a certain tool.
```bash
docker COMMAND --help
```
